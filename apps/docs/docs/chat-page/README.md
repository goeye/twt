# 聊天页面

除了在网站中嵌入小部件，你还可以通过独立聊天页面为客户提供咨询入口，适用于无需部署网站代码的场景。

## 两种接入方式

| 方式 | 说明 | 适用场景 |
| --- | --- | --- |
| [直接链接](./direct-link) | 使用 TWT Chat 托管的聊天页面链接 | 邮件签名、App 跳转、社交媒体分享 |
| [自托管部署](./self-hosted) | 下载 HTML 文件部署到自有服务器 | 需要自有域名访问、对品牌一致性有要求 |

## 自定义配置

两种方式均支持通过 URL 参数自定义语言和传递客户信息，详见 [URL 参数说明](./parameters)。
