# 聊天页面

无需在网站部署代码，通过独立聊天页面为客户提供咨询入口。

## 能做什么

- 使用 TWT Chat 托管的聊天页面链接，直接分享给用户
- 将 HTML 文件部署到自有服务器，使用自己的域名
- 通过 URL 参数设置界面语言和传递客户身份信息

## 怎么做

| 方式 | 说明 | 适用场景 |
| --- | --- | --- |
| [直接链接](./direct-link) | 使用 TWT Chat 托管的聊天页面链接 | 邮件签名、App 跳转、社交媒体分享 |
| [自托管部署](./self-hosted) | 下载 HTML 文件部署到自有服务器 | 需要自有域名、对品牌一致性有要求 |

两种方式均支持通过 [URL 参数](./parameters) 自定义语言和传递客户信息。

## 能获得什么

- 无需改动网站代码，即可为用户提供聊天入口
- 可嵌入邮件签名、App 内跳转、社交媒体主页等任意场景
- 自托管方案支持自有域名，保持品牌一致性
