# 自托管部署

下载包含聊天页面代码的 HTML 文件，上传到你自己的服务器或 OSS，通过自有域名访问。

## 部署步骤

### 1. 下载 HTML 文件

登录 TWT Chat 后台，进入 **设置 > 安装 > 聊天页面**，点击「下载 HTML 文件」。

下载的文件已包含你的应用配置（appid 等），无需额外修改即可使用。

### 2. 上传至服务器或 OSS

将 HTML 文件上传到你的 Web 服务器或 OSS，确保可通过 HTTP/HTTPS 访问。

### 3. 配置域名（可选）

将你的自有域名解析到服务器或 OSS，访客即可通过自有域名打开聊天窗口。

建议使用 HTTPS 部署，确保聊天数据传输安全。

## 自定义配置

自托管页面支持通过 URL 参数自定义语言和传递客户信息，详见 [URL 参数说明](./parameters)。
