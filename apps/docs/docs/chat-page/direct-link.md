# 直接链接

将聊天页面链接发给客户，打开即可开始对话。无需网站部署，适用于邮件、App、社交媒体等渠道。

## 链接格式

```
https://visitorchat.twt.com/direct/{你的项目APPID}
```

## 使用场景

- **邮件签名**：在客服邮件底部附上聊天链接
- **App 内嵌**：通过 WebView 打开聊天页面
- **社交媒体**：在公众号菜单、微博简介等位置放置链接
- **二维码**：将链接生成二维码供线下扫码使用

## 自定义配置

直接链接支持通过 URL 参数自定义语言和传递客户信息，详见 [URL 参数说明](./parameters)。
