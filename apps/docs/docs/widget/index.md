# 功能说明

在你的网站中嵌入 TWT Chat 聊天组件，让访客无需离开页面即可与客服实时沟通。

## 能做什么

- 一行代码嵌入聊天浮窗，无需额外依赖
- 支持英文、简体中文、繁体中文三种界面语言
- 将已登录用户的身份信息同步到客服工作台
- 通过 JavaScript API 程序化控制聊天窗口的开关、语言切换、用户登录

## 怎么做

1. [基础安装](./installation) — 获取安装代码，粘贴到页面 `</body>` 前
2. [配置项参考](./configuration) — 设置语言、隐藏默认图标、传递用户信息
3. [JavaScript API](./javascript-api) — 运行时控制聊天窗口

## 能获得什么

- 访客在你的网站上即可实时发起咨询，无需跳转
- 客服工作台中可查看访客的登录身份，提升服务效率
- 可与自有 UI 深度集成，完全自定义触发方式和外观
