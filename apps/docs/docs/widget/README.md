# 网站小部件

在你的网站中嵌入 TWT Chat 聊天组件，让访客无需离开页面即可与客服实时沟通。

## 功能特性

- 一行代码嵌入，无需额外依赖
- 支持自定义主题（亮色 / 暗色）
- 支持多语言（中文简体、中文繁体、英文）
- 提供 JavaScript API，可程序化控制聊天窗口
- 支持传递客户身份信息，实现登录用户识别
