# 在线聊天

在线聊天模块提供会话的创建、删除和管理接口。

## 接口列表

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| [创建单聊](./create-session) | POST | 创建单个聊天会话 |
| [创建单聊-批量](./create-session-batch) | POST | 批量创建单聊会话 |
| [创建群聊-批量](./create-group) | POST | 批量创建群聊会话 |
| [删除单聊](./delete-session) | POST | 删除单个聊天会话 |
| [解散群聊](./dissolve-group) | POST | 解散单个群聊 |
| [批量解散群聊](./dissolve-group-batch) | POST | 批量解散群聊 |
