# TWT Chat 开发者文档

在你的网站、App 或系统中接入在线客服，3 分钟完成部署。

## 选择接入方式

| 模块 | 适用场景 |
|------|----------|
| [网站小部件](/widget/) | 在任意网页嵌入聊天浮窗，一行代码完成接入 |
| [聊天页面](/chat-page/) | 独立聊天页面，适合插件、App、社交媒体场景 |

## 深度集成

| 模块 | 适用场景 |
|------|----------|
| [开放 API](/api/) | 服务端管理会话、同步客户数据 |
| [Webhooks](/webhooks/) | 实时接收聊天事件，驱动业务系统自动化 |

## 常见场景

| 我想… | 推荐方案 |
|-------|----------|
| 在网站右下角放一个聊天按钮 | [网站小部件 → 基础安装](/widget/installation) |
| 在原生 App 里放一个聊天按钮 | [聊天页面 → 直接链接](/chat-page/direct-link) |
| 用自己的域名部署客服窗口 | [聊天页面 → 自托管部署](/chat-page/self-hosted) |
| 识别访客的登录身份 | [网站小部件 → 配置访客参数](/widget/configuration) |
