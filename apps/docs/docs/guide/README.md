# 快速开始

## 能做什么

- 在任意网页嵌入聊天浮窗，访客无需离开页面即可咨询
- 将已登录用户的身份信息同步到客服工作台
- 支持简体中文、繁体中文、英文三种界面语言

## 怎么做

1. 登录 TWT Chat 后台，进入 **设置 > 安装 > 网站代码**，复制安装代码
2. 将代码粘贴到网站每个页面的 `</body>` 标签之前
3. 刷新页面，右下角出现聊天图标即接入成功

详细步骤见 [3 分钟接入](./quickstart)。

## 能获得什么

- 网站上线聊天浮窗，访客可实时发起咨询
- 客服在工作台中统一处理所有对话
- 后续可通过 [JavaScript API](/widget/javascript-api)、[开放 API](/api/)、[Webhooks](/webhooks/) 进行深度集成

## 前置条件

- 已注册 TWT Chat 账号并获取 `appid`
