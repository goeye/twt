# 概述

TWT Chat 是一款在线客服 SaaS 平台，提供多种接入方式帮助你快速为网站和应用添加实时聊天能力。

## 接入方式

| 方式 | 适用场景 | 说明 |
| --- | --- | --- |
| [网站小部件](/widget/) | 自有网站 | 在网页中嵌入聊天浮窗，访客无需离开页面即可发起咨询 |
| [聊天页面](/chat-page/) | 邮件、App、社交媒体 | 通过链接或自托管 HTML 打开独立聊天窗口 |
| [Webhooks](/webhooks/) | 业务系统集成 | 实时接收平台事件（如未回复消息），触发自动化流程 |
| [开放 API](/api/) | 深度集成 | 通过 REST API 创建会话、发送消息、管理客户数据 |

## 前置条件

- 已注册 TWT Chat 账号
- 已在后台创建应用并获取 `appid`
