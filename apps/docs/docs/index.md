# TWT Chat 开发者文档

在你的网站、App 或系统中接入在线客服，3 分钟完成部署。

## 接入方式

| 模块 | 适用场景 |
| --- | --- |
| [网站小部件](/widget/) | 在任意网页嵌入聊天浮窗，一行代码完成接入 |
| [聊天页面](/chat-page/) | 独立聊天页面，适合邮件、App、社交媒体场景 |

## Webhooks 与 API

| 模块 | 适用场景 |
| --- | --- |
| [Webhooks](/webhooks/) | 实时接收聊天事件，驱动业务系统自动化 |
| [开放 API](/api/) | 通过 REST API 创建会话、发送消息、管理客户数据 |

## 常见场景

| 我想... | 推荐方案 |
| --- | --- |
| 在网站右下角加一个聊天按钮 | [网站小部件 → 3 分钟接入](/widget/installation) |
| 在邮件或 App 里放一个聊天链接 | [聊天页面 → 直接链接](/chat-page/direct-link) |
| 用自己的域名部署聊天页 | [聊天页面 → 自托管部署](/chat-page/self-hosted) |
| 让客服看到访客的登录身份 | [网站小部件 → JavaScript API](/widget/javascript-api) |
| 服务端自动创建或关闭会话 | [开放 API → 功能说明](/api/) |
| 聊天事件触发 CRM 或工单 | [Webhooks → 功能说明](/webhooks/) |
