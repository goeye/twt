# 配置指南

在 TWT Chat 后台配置 Webhook，接收平台事件推送。

## 前置条件

1. 在 **开发设置** 中生成 `AppSecret`（用于签名校验）
2. 准备一个可公网访问的 HTTPS 接收地址

## 配置步骤

### 1. 生成 AppSecret

进入 **设置 > 开发设置**，生成 AppSecret。此密钥用于对 Webhook 请求进行签名校验。

::: warning 重要
请妥善保管 AppSecret，不要泄露到前端代码或公共仓库中。
:::

### 2. 填写 Webhook URL

进入 **设置 > Webhooks** 页面，输入接收事件通知的 URL 地址，点击「保存」。

::: tip 注意
保存 URL 前需先完成签名校验配置，否则无法保存。
:::

### 3. 订阅事件

在 Webhooks 页面选择需要接收的事件类型，开启对应的开关。

当前支持的事件详见 [事件类型](./events)。
