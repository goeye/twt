# PRD：在线会话分类筛选

## 1. 概述

### 1.1 功能定位

在会话模块的"在线会话"和"AI Agent"队列中，搜索栏下方增加按访客类型的分类筛选标签（全部 / 访客 / 客户），帮助客服快速定位目标会话。

### 1.2 功能目标

- 客服可按"访客"或"客户"类型快速筛选会话列表
- 每个分类标签实时显示对应会话数量，数据为空时显示 0
- 未接入客户标识的项目不展示分类标签，保持现有体验

### 1.3 功能入口

| 入口 | 位置 | 触发方式 |
|-----|------|---------|
| 分类筛选标签 | 会话列表搜索栏下方 | 点击对应标签切换 |

---

## 2. 分类标签定义

| 标签 | 含义 | 计数规则 |
|-----|------|---------|
| 全部 | 当前队列中的所有会话 | 当前队列会话总数 |
| 访客 | 未关联客户资料的匿名访客会话 | 当前队列中 tag 为"访客"的会话数 |
| 客户 | 已关联客户资料的会话（含 VIP） | 当前队列中 tag 为"客户"或"VIP"的会话数 |

- 默认选中"全部"标签
- 各标签后跟随实时计数，数据为空显示 0

---

## 3. 显示条件

### 3.1 适用队列

分类标签在以下队列中显示：

| 分组 | 队列 | 显示分类标签 |
|-----|------|------------|
| 在线会话 | 待回复 | 是 |
| 在线会话 | 排队中 | 是 |
| 在线会话 | 待处理 | 是 |
| 在线会话 | 已回复 | 是 |
| 在线聊天 | 聊天 | 否 |
| AI Agent | AI Agent | 是 |

### 3.2 前提条件

- 系统已接入客户标识时，显示"全部"、"访客"、"客户"三个标签
- 系统未接入客户标识时，仅显示"全部"标签，不显示"访客"和"客户"标签

---

## 4. 交互规则

### 4.1 筛选切换

| 操作 | 系统行为 |
|-----|---------|
| 点击"访客"标签 | 标签高亮，会话列表仅显示访客类型的会话 |
| 点击"客户"标签 | 标签高亮，会话列表仅显示客户类型的会话 |
| 点击"全部"标签 | 标签高亮，会话列表显示所有会话 |

### 4.2 筛选重置

| 场景 | 系统行为 |
|-----|---------|
| 切换到其他队列 | 分类筛选自动重置为"全部" |

### 4.3 与搜索联动

分类筛选与关键词搜索同时生效。例如选中"客户"标签后输入搜索关键词，列表仅显示匹配关键词的客户类型会话。

---

## 5. 测试要点

| 用例 | 操作步骤 | 预期结果 |
|-----|---------|---------|
| 默认状态 | 进入"待回复"队列 | 显示"全部 N"、"访客 X"、"客户 Y"三个标签，"全部"高亮 |
| 筛选访客 | 点击"访客"标签 | 列表仅显示访客类型会话，"访客"标签高亮 |
| 筛选客户 | 点击"客户"标签 | 列表仅显示客户类型会话，"客户"标签高亮 |
| 切回全部 | 点击"全部"标签 | 列表恢复显示所有会话 |
| 空数据计数 | 当前队列无客户类型会话 | "客户"标签显示"客户 0" |
| 切换队列重置 | 选中"访客"后切换到"排队中"队列 | 筛选重置为"全部" |
| 搜索联动 | 选中"客户"标签后输入搜索关键词 | 列表仅显示匹配关键词的客户会话 |
| AI Agent 队列 | 进入"AI Agent"队列 | 同样显示三个分类标签，筛选功能一致 |
| 在线聊天队列 | 进入"聊天"队列 | 仅显示"全部"标签，不显示"访客"和"客户" |
| 未接入客户标识 | 系统未接入客户标识 | 所有队列仅显示"全部"标签 |
