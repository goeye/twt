## 技术实现

### 统一的集成模式

所有应用都采用相同的集成模式：

```typescript
// main.ts
import { loadTenantConfig, provideTenant } from "@twt/branding";

async function bootstrap() {
  // 1. 加载租户配置
  const tenantConfig = await loadTenantConfig();
  
  // 2. 提供租户配置（Vue provide/inject）
  provideTenant(tenantConfig);
  
  // 3. 注入 CSS 变量
  document.documentElement.style.setProperty(
    "--xxx-color-brand-primary",
    tenantConfig.primaryColor
  );
  
  // 4. 设置页面标题和 favicon
  document.title = `${tenantConfig.displayName} - xxx`;
  favicon.href = tenantConfig.favicon;
  
  // 5. 挂载应用
  app.mount("#app");
}
```

### 组件中使用

```vue
<script setup lang="ts">
import { useTenant } from '@twt/branding';

const tenant = useTenant();
</script>

<template>
  <div>{{ tenant.displayName }}</div>
</template>
```
