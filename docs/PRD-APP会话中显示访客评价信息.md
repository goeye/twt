# PRD - APP 会话中显示访客评价信息

## 需求概述

在会话聊天详情、会话档案详情、会话信息中显示访客评价信息，与客服网页端保持一致。

## 功能范围

### 包含功能

- 会话聊天详情中显示评价消息
- 档案页面的会话记录列表中显示评价字段
- 会话信息页面中显示会话评价字段

## 交互规则

### 会话聊天详情中的评价显示

**评价消息布局**
- 评价消息使用系统消息样式
- 评价消息文案格式：`"访客提交了评价：{评价文本}"`
- 访客取消评价显示文案：`"访客取消了评价"`

**评价消息位置**
- 评价消息按时间顺序插入消息列表中
- 如果访客未评价，不显示评价消息

### 档案页面中的评价显示

**会话记录列表**
- 在会话档案详情中也需要同步展示访客评价系统消息

### 会话信息页面中的评价显示

**评价字段位置**
- 在"会话信息"标签页的"基础信息"区域中
- 常驻显示字段标签："会话评价"
- 字段位置：在"接待时间"字段下方

## 边界情况

**未评价状态**
- 如果访客未提交评价，会话聊天详情中不显示评价消息
