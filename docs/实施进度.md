## 如何测试

### 启动代理商管理后台

```bash
# 开发模式
pnpm --filter reseller-admin dev

# 构建
pnpm --filter reseller-admin build
```

访问：`http://localhost:5173`

### 测试品牌配置

1. 打开代理商管理后台
2. 查看左侧菜单显示的品牌名称（默认是 "TWT Chat"）
3. 点击"品牌配置"，修改品牌名称、颜色等
4. 保存后，主题色会动态更新

### 切换租户

修改 `packages/branding/src/loader.ts` 中的 `loadTenantConfig` 函数，返回不同的租户配置：

```typescript
// 开发环境：加载 demo 租户
if (import.meta.env.DEV) {
  const demoConfig = await import('./tenants/reseller-demo.json')
  return demoConfig.default
}
```

## 注意事项

### 关于 APP 独立打包的警告

你选择了"每个代理商独立 APP"，这会带来：

**成本（每个代理商）：**
- Apple Developer 账号：$99/年
- Google Play 账号：$25 一次性
- 开发者时间：首次打包 2-3 天，每次更新 1 天

**如果有 10 个代理商：**
- 年度账号成本：$990 + $250 = $1,240
- 维护 20 个 APP（10 个 iOS + 10 个 Android）
- 每次更新需要打包 20 次，上架 20 次
- 应用商店可能拒绝"马甲包"（4.3 Spam 条款）

**建议：**
- 短期（MVP）：只做 Web 白标，暂不做 APP
- 中期：做统一 APP + 工作空间切换，验证需求
- 长期：如果代理商愿意承担成本，再做独立 APP

### 关于商业模式验证

"代理商自主定价 + 按订单百分比分成"这个模式还没跟代理商确认。建议：
1. 先跟代理商确认商业模式
2. 确认是否真的需要"所有前端白标"
3. 确认是否真的需要"独立 APP"
4. 签署意向书，明确需求和时间表
