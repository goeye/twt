## 页面功能详情

### 1. 订单管理
- 统计：总订单数、总收入、待结算、已结算
- 筛选：状态、套餐、搜索
- 列表：订单号、项目、套餐、金额、佣金、状态、时间
- 操作：查看详情、导出订单

### 2. 项目管理
- 统计：总项目数、总客服数、总会话数、平均响应时间
- 筛选：状态、套餐、搜索
- 列表：项目名称、客户、套餐、客服数、会话数、状态
- 操作：查看项目详情

### 3. 佣金管理
- 统计：累计佣金、待结算、已提现、本月佣金
- 提现：输入金额、选择账户、提交申请
- 明细：按状态筛选（全部、待结算、已结算、已提现）
- 记录：提现单号、金额、手续费、实际到账、状态

### 4. 官网配置
- 首页：标题、副标题
- 功能：添加/删除功能列表（标题、描述、图标）
- 页脚：公司名称、链接列表
- 操作：重置、保存

### 5. 邮件管理
- SMTP：服务器、端口、用户名、密码、测试连接
- 发件人：名称、邮箱、回复邮箱
- 批量发送：收件人列表、主题、内容、预览、发送
- 记录：主题、收件人数、成功/失败数、状态、时间

### 6. OSS 管理
- 统计：存储空间、文件数量、流量使用、请求次数
- 配置：提供商、Bucket、区域、密钥、测试连接
- 文件：列表、查看、删除、上传

### 7. 域名绑定
- 列表：域名、DNS 状态、SSL 状态、添加时间
- 添加：输入域名、自动生成 CNAME 记录
- DNS：配置说明、验证功能
- 操作：删除域名
