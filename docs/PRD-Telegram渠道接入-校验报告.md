#### E.2 欢迎消息超长处理规则缺失

**位置**：3.3.1 - Bot 配置管理

**问题**：欢迎消息最大长度 500 字符，但未说明超出时如何处理

**建议**：补充规则："欢迎消息输入时限制最多 500 字符，超出时禁止继续输入"

---

#### E.3 重复 Bot Token 处理规则缺失

**位置**：3.3.1 - Bot 配置管理

**问题**：未说明添加重复 Bot Token 时如何处理（允许/拒绝/提示）

**建议**：补充规则："同一 Bot Token 不允许重复添加，保存时校验重复则提示'该 Bot 已存在'"

---

#### E.4 删除 Bot 后现有会话处理规则缺失

**位置**：3.3.1 - Bot 配置管理

**问题**：删除 Bot 后，该 Bot 的历史会话如何处理？是否仍可查看？是否可继续回复？

**建议**：补充规则："删除 Bot 后，历史会话保留可查看，但无法继续发送消息，输入框显示'Bot 已删除，无法发送消息'"

---

#### E.5 同一用户多次发消息会话合并规则缺失

**位置**：3.3.2 - Telegram 会话接收

**问题**：同一 Telegram 用户多次发消息时，是创建新会话还是合并到已有会话？已结束的会话如何处理？

**建议**：补充规则："同一 Telegram User ID 的消息合并到同一会话。若会话已结束，新消息自动重新开启会话并重新分配"

### 🟡 中等问题（建议修复）

#### B.1 删除 Bot 弹窗按钮文案不符合全局规范

**位置**：3.3.1 - Bot 配置管理

**问题**：删除弹窗按钮为"取消、确认结束"，但根据全局规范，删除操作应为"取消、删除"

**建议**：修改为："按钮：取消（ghost）、删除（红色背景 #ef4444）"

---

#### B.2 编辑 Bot 成功 Toast 文案不符合全局规范

**位置**：3.3.1 - Bot 配置管理

**问题**：编辑成功提示"Bot 更新成功"，但根据全局规范应为"编辑成功"

**建议**：修改为："编辑成功提示'编辑成功'，新增成功提示'新增成功'"

---

#### B.3 结束会话弹窗按钮文案不一致

**位置**：3.3.5 - 会话结束与重新开启

**问题**：弹窗按钮为"取消、确认结束"，但其他地方（如 Email）使用"取消、确认结束"，需确认是否一致

**建议**：确认全局规范，统一为"取消、确认结束"或"取消、结束"

---

#### C.1 Bot Username 字段来源不明确

**位置**：3.3.1 - Bot 配置管理

**问题**：Bot 列表显示"Bot Username"，但输入表单中未要求填写，来源不明（是从 Telegram API 获取还是用户输入？）

**建议**：补充说明："Bot Username 从 Telegram API 自动获取，保存 Bot Token 时调用 getMe 接口获取"

---

#### D.1 Bot 状态定义不完整

**位置**：3.3.1 - Bot 配置管理

**问题**：Bot 状态只定义了"正常/异常"，但未说明何时变为异常、异常后如何恢复

**建议**：补充状态机："正常：Webhook 连接正常 → 异常：Webhook 连接失败或 Token 失效 → 恢复：重新验证 Token 或修复 Webhook"

---

#### D.2 会话状态定义不完整

**位置**：4. 数据模型

**问题**：TelegramSession.status 字段类型为 string，但未定义具体枚举值（pending/processing/resolved/closed？）

**建议**：补充枚举值："status: 'pending' | 'processing' | 'resolved' | 'closed'，含义详见会话管理规则"

---

#### E.6 消息发送失败后重试次数限制缺失

**位置**：3.3.3 - 消息发送

**问题**：消息发送失败显示重试按钮，但未说明可重试次数、重试间隔、多次失败后如何处理

**建议**：补充规则："重试无次数限制，每次重试间隔 2 秒，连续失败 3 次后提示'消息发送失败，请稍后重试'"

---

#### E.7 文件上传大小限制缺失

**位置**：3.3.3 - 消息发送

**问题**：支持文件上传，但未说明文件大小限制、格式限制

**建议**：补充规则："图片最大 10MB，支持 jpg/png/gif；文档最大 20MB，支持常见格式；超出时提示'文件过大，最大支持 XMB'"

#### F.1 筛选面板渠道选项未更新
