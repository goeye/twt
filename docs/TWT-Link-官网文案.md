## 七、FAQ

**Q：TWT Link 和微信企业版有什么区别？**
A：TWT Link 支持私有化部署，数据完全存储在企业自有服务器，不依赖任何第三方平台，适合对数据安全有严格要求的企业。

**Q：部署需要多久？**
A：支持 Docker / K8s 一键部署，技术团队通常可在半天内完成上线。

**Q：支持哪些平台？**
A：iOS、Android、Windows、macOS 全端支持，多设备同时在线。

**Q：消息加密是怎么实现的？**
A：采用端对端加密，消息在发送端加密、接收端解密，服务端无法读取消息内容。

**Q：管理员能控制哪些权限？**
A：支持禁止指定成员私聊、设置部分频道/群组对特定成员不可见、强制退出、禁言、账号注销等。
